## Contexto

La auditoria ciega de resolucion ya no se cae cuando una jornada madura pero el entorno no puede consultar el archivo meteorologico remoto. En la corrida del `7 de abril de 2026`, los eventos del snapshot `2026-04-05` pasaron de pendientes a maduros, pero no pudieron puntuarse porque este workspace no tiene observaciones locales para esas estaciones y la API de archivo no estuvo accesible.

## Decisiones tecnicas

- El runner intenta primero resolver la temperatura real desde `weather_trading.db`.
- Si no hay observacion local y falla la consulta remota del archivo, el evento se marca como omitido en lugar de abortar toda la auditoria.
- Tras el primer fallo remoto, el runner deja de reintentar para el resto de eventos de la corrida y conserva el error en el snapshot.
